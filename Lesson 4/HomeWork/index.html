PHP и JavaScript

Языки програмирования PHP и JavaScript позволяет сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно сделать красивую менюшку или слайдер

Виды скриптов

Для этого пишуться скрипты (англ. script - сценарий) - программы позволяющие реагировать на лействия пользователя. Скрипты бывают двух видов:

  • те, которые выполняются на сервере, а результат их выполнения приходит в браузеру прльзователю ужк в готовом виде.Это скрипты, написаные на языке PHP . На нем пишуться CMS-ки - это системы управления контентом
  • те, которые выполняются прямо в браузере пользователя. Это скрипты, написаные на языке JavaScript . Они чаще использубться для, того чтобы слеоать страницу более удобной и красивой
